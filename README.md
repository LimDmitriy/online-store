# Учебный проект onlile-store

## Онлайн-магазин на Django с Bootstrap

Это пример онлайн-магазина, созданного с использованием **Django 5.2.6**, **Python 3.12** и **Bootstrap 5**.  
Проект поддерживает базовые функции: домашняя страница, контактная страница .

---

## Особенности

- Использование **Bootstrap 5** для адаптивного дизайна.
- Шаблоны Django. 
- Структура проекта готова для дальнейшего расширения
