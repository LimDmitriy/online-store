# Учебный проект onlile-store

## Онлайн-магазин на Django с Bootstrap

Это пример онлайн-магазина, созданного с использованием **Django 5.2.6**, **Python 3.12** и **Bootstrap 5**.  
Проект поддерживает базовые функции: домашняя страница, контактная страница .

---

## Особенности

- Использование **Bootstrap 5** для адаптивного дизайна.  
- Поддержка **Django шаблонов** с базовым шаблоном `base.html`, включающим шапку, подвал, стили и скрипты.  
- Подшаблон для **главного меню `inc_menu.html`**, подключаемый через `{% include %}`, что уменьшает повторение кода.  
- **Циклы Django** для отображения списков товаров с автоматическим обрезанием текста для карточек.  
- Гибкая структура шаблонов для **легкого расширения и изменения дизайна** страниц.  
- Подключена **PostgreSQL**.  
- Реализованы модели **Category** и **Product**.  
- Настроены **миграции** и **фикстуры**.  
- Модели зарегистрированы в **админке**.  
- Настроены **фильтры и поиск** в админке.  
- Создана **кастомная команда** для наполнения базы тестовыми данными.  
- Структура проекта готова для дальнейшего расширения.
